# Data Cleaning & Structural Validation

## Project Overview

This project demonstrates the process of transforming a raw, unstructured dataset into a clean, consistent, and analysis-ready dataset using Microsoft Excel.

The dataset contained multiple data quality issues, including duplicate records, missing values, inconsistent text formatting, unnecessary spaces, and structural inconsistencies. The objective was to improve data quality while preserving the integrity of the original information.

---

## Objectives

- Identify and remove duplicate records
- Handle missing values appropriately
- Standardize text formatting
- Validate numerical and percentage fields
- Ensure consistency across the dataset
- Produce a reliable dataset ready for analysis

---

## Dataset Issues Identified

The raw dataset contained several common data quality problems:

- Duplicate records
- Missing payment information
- Missing revenue values
- Inconsistent capitalization
- Extra spaces in contact names
- Blank rows
- Formatting inconsistencies
- Formula validation requirements

---

## Data Cleaning Process

### 1. Dataset Inspection

- Examined all columns
- Reviewed data types
- Identified missing values
- Checked for duplicates
- Reviewed formulas

---

### 2. Duplicate Removal

- Identified duplicate records
- Removed exact duplicate rows while preserving original data integrity

---

### 3. Missing Value Handling

- Filled missing payment values where appropriate
- Handled missing revenue values without introducing incorrect information

---

### 4. Text Standardization

Cleaned text fields by:

- Removing leading spaces
- Removing trailing spaces
- Removing multiple spaces
- Converting names into Proper Case

Example:

Before

```
     BILL    smITH
```

After

```
Bill Smith
```

---

### 5. Formula Validation

Verified Profit Margin calculations.

Formula used:

Profit Margin = Profit / Revenue

Validated all rows for calculation consistency.

---

### 6. Data Validation

Verified:

- No duplicate records remain
- Contact names follow a consistent format
- Blank rows removed
- Percentage formatting applied
- Dataset structure preserved
